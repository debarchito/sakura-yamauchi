# Sakura Yamauchi

A Discord bot created in the memory of [Sakura Yamauchi](https://i-want-to-eat-your-pancreas.fandom.com/wiki/Sakura_Yamauchi) (Kanji: 山内桜良).

## Plans

A general purpose feature-rich bot with music, moderation, games, etc.

## Want to contribute?

Sure! Contributions are always welcomed! Just take a moment to go through the [STYLEGUIDE](https://github.com/debarchito/sakura-yamauchi/blob/main/STYLEGUIDE.md) to ensure your code style matches that of the project.
